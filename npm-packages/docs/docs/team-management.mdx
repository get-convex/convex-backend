Convex provides administrative features for controlling how members access your
team, projects, and deployments:

- **[Single Sign-On (SSO)](/team-management/sso.md)** — authenticate team
  members through your organization's identity provider.
- **[Custom Roles](/team-management/custom-roles.mdx)** — define fine-grained
  permission policies beyond the built-in Admin and Developer roles.
- **[Role Actions](/team-management/role-actions.mdx)** — reference for every
  action a role can grant, and which built-in roles cover which actions.

For an overview of the built-in team and project roles, see
[Roles and permissions](/dashboard/teams/teams.mdx#roles-and-permissions).
