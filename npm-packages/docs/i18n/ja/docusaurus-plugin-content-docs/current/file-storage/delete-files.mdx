Convex に保存されているファイルは、
[ミューテーション](/functions/mutation-functions.mdx)、
[アクション](/functions/actions.mdx)、および
[HTTP アクション](/functions/http-actions.mdx) から、
ストレージ ID を受け取る
[`storage.delete()`](/api/interfaces/server.StorageWriter#delete) 関数を使って削除できます。

ストレージ ID は `"_storage"` システムテーブル内のドキュメントに対応します（[メタデータ](/file-storage/file-metadata.mdx) を参照）。そのため、
`v.id("_storage")` を使って検証できます。
