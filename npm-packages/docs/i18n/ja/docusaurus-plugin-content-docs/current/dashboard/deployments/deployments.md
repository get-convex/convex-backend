Convex の各プロジェクトにはメインとなる本番デプロイメントがあり、チーム内の各開発者は
自分専用の開発デプロイメントをセットアップすることもできます。
さらに、本番にデプロイする前にバックエンドの変更をテストするために使う
[プレビューデプロイメント](/production/hosting/preview-deployments.mdx) も用意されています。

[デプロイメントページ](https://dashboard.convex.dev/deployment) では、
ページ左上のドロップダウンメニューを使って、本番デプロイメント、自分の開発デプロイメント、
および任意のプレビューデプロイメントを切り替えることができます。
