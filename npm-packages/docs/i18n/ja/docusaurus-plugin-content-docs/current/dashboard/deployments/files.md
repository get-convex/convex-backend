[ファイルストレージページ](https://dashboard.convex.dev/deployment/files)には、
[デプロイメントに保存されているファイル](/file-storage.mdx)が表示されます。このページでは、
各ファイルのストレージ ID、サイズ、コンテンツタイプも確認できます。新しいファイルをアップロードしたり、
既存のファイルをダウンロードまたは削除したりできます。

ストレージ ID は、データベース内のドキュメントから参照されている場合があります。

<Admonition type="tip">

新しいファイルをアップロードすると、UI には直近にアップロードしたファイルの名前が表示されます。
ただし、これらの名前は永続化されないため、ページを再読み込みすると表示されなくなります。

</Admonition>
