![History Dashboard Page](/screenshots/history.png)

<Admonition type="info">

履歴ページは Convex Professional プランでのみ利用できます。

</Admonition>

この[履歴ページ](https://dashboard.convex.dev/deployment/history)は、選択した
デプロイメントで発生した構成関連イベントの監査ログです。関数のデプロイ、
インデックスの変更、環境変数の変更などが含まれます。

チームに関連するイベントの監査ログは、
[チーム監査ログ](/dashboard/teams.md#audit-log) からも確認できます。
