<span className="convex-hero">
  Convex Components はコードとデータをサンドボックス環境にまとめ、バックエンドに新機能を自信を持って素早く追加できるようにします。
</span>

Convex Components は、小さな自己完結型の Convex バックエンドのようなもので、インストールしても常に安全です。明示的に渡さない限り、アプリのテーブルを読み取ったり、アプリの関数を呼び出したりすることはできません。

全体的なビジョンについては、
[Convex: The Software-Defined Database](https://stack.convex.dev/the-software-defined-database#introducing-convex-components)
を参照してください。

コンポーネントは npm から、またはローカルフォルダーからインストールできます。インストールされると、それぞれが独自のデータベーステーブルと分離された関数実行環境を持ちます。
コンポーネントの完全なディレクトリは
[Convex website](https://convex.dev/components)
で確認できます。
