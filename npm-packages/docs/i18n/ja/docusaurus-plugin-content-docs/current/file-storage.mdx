ファイルストレージを使うと、アプリでのファイルアップロードの実装、サードパーティ API から受け取ったファイルの保存やサードパーティ API へのファイル送信、そしてユーザーへの動的なファイル配信を簡単に行えます。すべてのファイル形式をサポートしています。

- Convex にファイルを保存し、データベースドキュメントから参照できるようにファイルを[アップロード](/file-storage/upload-files.mdx)する
- サードパーティ API から生成または取得したファイルを[保存](/file-storage/store-files.mdx)する
- URL 経由でファイルを[配信](/file-storage/serve-files.mdx)する
- Convex に保存されているファイルを[削除](/file-storage/delete-files.mdx)する
- ファイルの[メタデータ](/file-storage/file-metadata.mdx)にアクセスする

保存されたファイルは
[ダッシュボード](/dashboard/deployments/files.md)
から管理できます。

**例:**
[HTTP アクションを使ったファイルストレージ](https://github.com/get-convex/convex-demos/tree/main/file-storage-with-http),
[クエリとミューテーションを使ったファイルストレージ](https://github.com/get-convex/convex-demos/tree/main/file-storage)
