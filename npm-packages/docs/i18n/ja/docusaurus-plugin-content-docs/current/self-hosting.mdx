セルフホスティングに興味がある方は、Convex のバックエンドを自分のサーバー上で実行できます。セルフホスト版 Convex は
[オープンソースのバックエンド](https://github.com/get-convex/convex-backend) を実行しており、
クラウドサービスが使用しているものと同じ最新のコードが常に含まれています。

セルフホスティングを始めるには、セルフホスティングガイドに従ってください:

<CardLink
  className="convex-hero-card"
  item={{
  href: "https://github.com/get-convex/convex-backend/blob/main/self-hosted/README.md",
  label: "セルフホスティングガイド",
}}
/>

セルフホスティングのサポートが必要な場合は、
[Discord コミュニティ](https://convex.dev/community) の `#self-hosted` チャンネルに参加してください。

セルフホスティングはすべての人に向いているわけではありません。より運用負荷の少ないソリューションを求めている場合は、
[Convex のホスティングサービス](https://convex.dev/pricing) の利用をおすすめします。

## オープンソースの Convex バックエンド \{#open-source-convex-backend\}

バックエンドの大部分は Rust で実装されており、サーバーサイド関数の実行環境を支える TypeScript も多用されています。

Convex におけるオープンソース化の詳細は、
[アナウンス](https://news.convex.dev/convex-goes-open-source/) と
[Software Engineering Daily ポッドキャスト](https://softwareengineeringdaily.com/2024/03/20/going-open-source-at-convex-with-james-cowling/) でご覧いただけます。
このソフトウェアは [FSL Apache 2.0 License](https://fsl.software/) の条件のもとで公開されています。

## その他の Convex オープンソースプロジェクト \{#other-convex-open-source-projects\}

Convex のバックエンド、クライアントライブラリ、ダッシュボード、CLI はすべてオープンソースです。
すべての内容は
[Convex の GitHub ページ](https://github.com/get-convex)
で確認できます。

### Convex クライアント \{#convex-clients\}

すべての Convex クライアントはオープンソースとして公開されています。

* [Convex JavaScript/TypeScript クライアント &amp; CLI](https://github.com/get-convex/convex-js)
* [Convex Python クライアント](https://github.com/get-convex/convex-py)
* [Convex Rust クライアント](https://github.com/get-convex/convex-rs)

### その他いろいろ \{#much-much-more\}

Convex はこのほかにも多くの便利なプロジェクトをオープンソースとして公開しています。[helpers](https://github.com/get-convex/convex-helpers)、[templates](https://github.com/orgs/get-convex/repositories?type=all\&q=template)、[demos](https://github.com/get-convex/convex-demos)、[testing harness](https://github.com/get-convex/convex-test) など、まだまだたくさんあります。すべての公開リポジトリの一覧は
[GitHub 上](https://github.com/orgs/get-convex/repositories?type=all)で確認できます。
