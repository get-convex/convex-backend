Convex は急速に進化しているプラットフォームであり、皆さまからのフィードバックを常にお待ちしています。

## フィードバックとサポート \{#feedback-and-support\}

一般的な質問、機能リクエスト、製品に関するフィードバックは、
[Convex Discord Community](https://convex.dev/community) にぜひお寄せください。特に、
みなさんが Convex 上でどのようなものを構築されるのかを見るのを楽しみにしています。

Discord チャンネルだけでは十分に対応できない、より具体的なサポートに関するご質問は、
[support@convex.dev](mailto:support@convex.dev) までお問い合わせください。

## Convex をフォローする \{#following-convex\}

リリースノートは [Convex News](https://news.convex.dev/tag/releases)
および [Convex Discord Community](https://convex.dev/community) で公開しています。

製品に関するお知らせ、記事、デモは
[Stack](https://stack.convex.dev/)、[News](https://news.convex.dev/)、
[Convex の YouTube チャンネル](https://www.youtube.com/channel/UCoC_9mdiPwIu1sDxDtGQggQ)、
そして [X（旧 Twitter）](https://x.com/convex) に掲載しています。

## 脆弱性の報告 \{#vulnerability-disclosure\}

Convex のセキュリティ上の問題を発見されたと思われる場合は、
[security@convex.dev](mailto:security@convex.dev) までご連絡ください。24 時間以内にご返信いたします。
こちらで問題に対処するまで、その内容を公開されないようご協力をお願いします。
