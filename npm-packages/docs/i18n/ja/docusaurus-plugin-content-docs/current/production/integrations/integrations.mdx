Convex は、ログストリーミングや例外レポートのためにサポートされているさまざまなサードパーティツールと連携します。

* [Log Streams](/production/integrations/log-streams) を使用すると、Convex デプロイメントからのログイベントを、Axiom、Datadog、またはカスタム webhook などのサポートされている送信先にストリーミングできます。
* [Exception Reporting](/production/integrations/exception-reporting) を使用すると、Convex 関数の実行時に発生するエラーを把握できるようになります。

## インテグレーションの設定 \{#configuring-an-integration\}

インテグレーションを設定するには、
Convex ダッシュボードの
[Deployment Settings](https://dashboard.convex.dev/deployment/settings) ページに移動し、
サイドバーの「Integrations」タブを開きます。このページには、
設定済みインテグレーションの一覧、それぞれの現在の正常性ステータス、
および新たに設定可能なインテグレーションが表示されます。新しくインテグレーションを設定するには、
対象のカードをクリックして、セットアップ手順に従ってください。

![Integrations Page](/screenshots/integrations_page.png)

## インテグレーションの削除 \{#deleting-an-integration\}

インテグレーションを削除して、設定済みの送信先へのイベント送信を停止するには、
設定済みパネルの右上隅にあるメニューアイコンを選択し、「Delete integration」を選択します。
確認後、数秒以内にそのインテグレーションは停止します。

[Convex Components Directory](https://convex.dev/components) を参照してください。
