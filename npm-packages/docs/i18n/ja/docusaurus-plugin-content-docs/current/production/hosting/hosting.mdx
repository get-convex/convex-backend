フルスタック Web アプリを公開する最も簡単な方法の一つは、
[Vercel](https://vercel.com) や [Netlify](https://netlify.com) といったホスティングプロバイダーを利用することです。

Vercel と Netlify はどちらも Git と連携しており、新しいリビジョンがプッシュされるたびにコードをデプロイします。アプリをホストするには、次の手順に従います。

1. すべてのファイルをコミットし、
   [GitHub](https://github.com/)、[GitLab](https://gitlab.com/)、[Bitbucket](https://bitbucket.org/) など好みの Git ホスティングプロバイダーにプッシュします。

2. 以下の該当するガイドに従ってください。

Netlify や Vercel を使用していない場合は、Custom Hosting のガイドに従ってください。

- [Vercel](/production/hosting/vercel.mdx)
- [Netlify](/production/hosting/netlify.mdx)
- [Custom Hosting](/production/hosting/custom.mdx)
