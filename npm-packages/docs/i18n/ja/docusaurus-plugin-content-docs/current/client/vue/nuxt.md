[Nuxt](https://nuxt.com/) は、Vue をベースにした高機能な Web フレームワークです。

コミュニティ主導でメンテナンスされている
[`convex-nuxt` npm package](https://www.npmjs.com/package/convex-nuxt) は、
Convex を Nuxt 3 エコシステムに深く統合するための機能を提供します。

内部的には [convex-vue](../vue.md) を使用しています。

使い始めるには [Nuxt Quickstart](/quickstart/nuxt.mdx) を参照してください。
より詳しいドキュメントは
[convex-nuxt GitHub page](https://github.com/chris-visser/convex-nuxt) にあります。

<Admonition type="info">

[`convex-nuxt` library](https://github.com/chris-visser/convex-nuxt/tree/main)
はコミュニティ主導でメンテナンスされています。このプロジェクトに取り組んでいる
メンテナーの [Chris Visser](https://github.com/chris-visser) に感謝します。

</Admonition>
