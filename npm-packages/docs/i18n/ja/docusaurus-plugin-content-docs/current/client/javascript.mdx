# Convex JavaScript クライアント \{#convex-javascript-clients\}

Convex アプリケーションは、[`fetch`](https://developer.mozilla.org/en-US/docs/Web/API/fetch) または
[`WebSocket`](https://developer.mozilla.org/en-US/docs/Web/API/WebSocket) を実装している
Node.js やあらゆる JavaScript ランタイムからアクセスできます。リアクティブな
[Convex Client](/api/classes/browser.ConvexClient) は、ウェブアプリケーションや長時間稼働する
Node.js サーバーが Convex クエリの更新を購読できるようにします。一方、
[Convex HTTP client](/api/classes/browser.ConvexHttpClient) は、通常サーバーサイドレンダリング、
マイグレーション、管理スクリプト、およびサーバーレス関数から、ある時点のクエリを一度きりで実行するために
使用されます。

React を使用している場合は、[React](/client/react.mdx) で説明している
専用の [`ConvexReactClient`](/api/classes/browser.ConvexClient) を参照してください。

## Convex クライアント \{#convex-client\}
