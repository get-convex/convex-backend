[Vue](https://vuejs.org/) は「手軽に使え、高性能で汎用的な Web ユーザーインターフェースを構築するためのフレームワーク」です。

コミュニティによってメンテナンスされている
[`convex-vue`](https://www.npmjs.com/package/convex-vue) npm パッケージは、Convex 向けの Vue.js
連携ライブラリです。また、[convex-nuxt](./vue/nuxt.md) の基盤にもなっています。

始めるには [Vue クイックスタート](/quickstart/vue.mdx) を参照してください。より詳しいドキュメントについては
[convex-vue GitHub ページ](https://github.com/chris-visser/convex-vue) を参照してください。
