[React Native](https://reactnative.dev/) で Convex を利用するには、
[Convex React クライアントライブラリ](/client/react.mdx) を使用します。

React Native 固有の設定については、
[React Native クイックスタート](/quickstart/react-native.mdx) を参照してください。

また、動作確認済みの
[Convex React Native デモ](https://github.com/get-convex/convex-demos/tree/main/react-native)
をクローンすることもできます。
