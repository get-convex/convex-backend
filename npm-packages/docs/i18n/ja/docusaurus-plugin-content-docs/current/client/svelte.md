Convex は Svelte のようなリアクティブな UI フレームワークと非常に相性が良いプラットフォームです。
[convex-svelte npm パッケージ](https://www.npmjs.com/package/convex-svelte) は、
Svelte 5 における宣言的なサブスクリプション機能を通じて
[ConvexClient](/api/classes/browser.ConvexClient) を拡張します。

使い始めるには [Svelte クイックスタート](/quickstart/svelte.mdx) を参照してください。Svelte
クライアントはオープンソースで、
[GitHub](https://github.com/get-convex/convex-svelte) で利用できます。
