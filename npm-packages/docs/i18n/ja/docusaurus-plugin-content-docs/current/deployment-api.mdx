Convex デプロイメントのパブリックインターフェースは、`convex` フォルダー内のファイルに定義された関数によって決まります。

各 Convex デプロイメントのパブリックな HTTP エンドポイントは、[HTTP アクション](/functions/http-actions.mdx) によって定義されるカスタム HTTP エンドポイントと、静的な [パブリック HTTP API](/http-api/index.md) で構成されます。

デプロイメントは、そのデプロイメントの管理者のみが利用できるプライベートエンドポイントも提供します:

- [ストリーミング エクスポート API](/streaming-export-api.mdx)
- [ストリーミング インポート API](/streaming-import-api.mdx)
- [プラットフォーム API](/deployment-platform-api.mdx)
