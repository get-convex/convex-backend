Convex からデータをエクスポートするには、
[バックアップを取得し](/database/backup-restore)、それを zip ファイルとしてダウンロードします。

あるいは、同じデータを
[コマンドライン インターフェース (CLI)](/cli.md#export-data-to-a-file)からエクスポートすることもできます。

```sh
npx convex export --path ~/Downloads
```
