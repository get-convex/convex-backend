Convex はオープンソースのリアクティブなデータベースであり、クエリは TypeScript コードとして
データベース内で直接実行されます。React コンポーネントが状態の変化に
反応するのと同様に、Convex のクエリはデータベースの変化に反応します。

Convex はデータベース、サーバー関数を書く場所、そしてクライアントライブラリを提供します。
これにより、動的でライブ更新されるアプリを簡単に構築・スケールできます。
