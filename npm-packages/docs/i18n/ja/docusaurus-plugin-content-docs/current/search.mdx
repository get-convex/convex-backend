RAG 対応のチャットボットでも、アプリケーション内のクイック検索でも、
Convex は強力な AI・検索機能を備えたプロダクトを簡単に構築するためのシンプルな API を提供します。

[ベクター検索](/search/vector-search.mdx) を使うと、ドキュメントの意味的な類似性に基づいて検索できます。ベクター埋め込みを用いて類似度を計算し、指定したクエリに類似したドキュメントを取得します。ベクター検索は、RAG のような一般的な AI 手法の重要な要素です。

[全文検索](/search/text-search.mdx) を使うと、ドキュメント内でキーワードやフレーズを検索できます。プレフィックスマッチをサポートしているため、タイプアヘッド検索を実現できます。Convex の全文検索も、他のすべての Convex クエリと同様にリアクティブで常に最新の状態に保たれるため、信頼性の高いクイック検索ボックスを簡単に構築できます。

[Convex アクション](functions/actions) を使うと、AI API を呼び出したり、データベースにデータを保存したり、ユーザーインターフェイスを制御したりできます。これを利用して[高度な AI アプリケーションを構築する方法の例](https://stack.convex.dev/tag/AI)も参照してください。
