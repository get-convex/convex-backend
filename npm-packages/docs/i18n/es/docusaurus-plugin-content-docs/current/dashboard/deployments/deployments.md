Cada proyecto en Convex tiene un despliegue principal de producción, y cada
desarrollador de tu equipo también puede configurar su propio despliegue de
desarrollo personal. Además, hay
[despliegues de vista previa](/production/hosting/preview-deployments.mdx) que se
usan para probar cambios en el backend antes de desplegarlos en producción.

Mientras estés en una [página de despliegue](https://dashboard.convex.dev/deployment),
puedes cambiar entre producción, tu despliegue de desarrollo y cualquier
despliegue de vista previa usando el menú desplegable en la esquina superior
izquierda de la página.
