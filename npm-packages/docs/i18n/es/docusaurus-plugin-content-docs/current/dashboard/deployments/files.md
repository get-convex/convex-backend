La [página de almacenamiento de archivos](https://dashboard.convex.dev/deployment/files) muestra
[los archivos almacenados en tu despliegue](/file-storage.mdx). La página también muestra los
ID de almacenamiento de los archivos, su tamaño y el tipo de contenido. Puedes subir archivos nuevos y
descargar o eliminar archivos existentes.

Los ID de almacenamiento pueden ser referenciados por documentos en tu base de datos.

<Admonition type="tip">

Cuando se suben archivos nuevos, la interfaz de usuario hará referencia al nombre del archivo recién
subido. Sin embargo, estos nombres no se persisten y dejarán de mostrarse
cuando se vuelva a cargar la página.

</Admonition>
