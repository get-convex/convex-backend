El almacenamiento de archivos facilita implementar la subida de archivos en tu app, almacenar archivos
provenientes de APIs de terceros y enviarles archivos, y servir archivos dinámicos a tus
usuarios. Se admiten todos los tipos de archivos.

- [Cargar](/file-storage/upload-files.mdx) archivos para almacenarlos en Convex y
  hacer referencia a ellos en los documentos de tu base de datos
- [Almacenar](/file-storage/store-files.mdx) archivos generados u obtenidos de
  APIs de terceros
- [Servir](/file-storage/serve-files.mdx) archivos a través de una URL
- [Eliminar](/file-storage/delete-files.mdx) archivos almacenados en Convex
- Acceder a los [metadatos](/file-storage/file-metadata.mdx) de los archivos

Puedes gestionar tus archivos almacenados en el
[panel de control](/dashboard/deployments/files.md).

**Ejemplos:**
[Almacenamiento de archivos con acciones HTTP](https://github.com/get-convex/convex-demos/tree/main/file-storage-with-http),
[Almacenamiento de archivos con consultas y mutaciones](https://github.com/get-convex/convex-demos/tree/main/file-storage)
