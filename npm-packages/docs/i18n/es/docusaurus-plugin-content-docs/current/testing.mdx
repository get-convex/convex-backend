Convex facilita la realización de pruebas en tu aplicación, mediante tests automatizados que se ejecutan en JavaScript o contra un backend real, y manualmente en los entornos dev, preview y staging.

## Pruebas automatizadas \{#automated-tests\}

### Biblioteca `convex-test` \{#convex-test-library\}

[Usa la biblioteca `convex-test`](/testing/convex-test.mdx) para probar tus funciones
de JavaScript con el excelente framework de pruebas Vitest.

### Pruebas con un backend real \{#testing-against-a-real-backend\}

Las versiones de código abierto de Convex te permiten probar toda tu lógica de backend ejecutándose en
un [backend local real de Convex](/testing/convex-backend.mdx).

### Configurar pruebas en CI \{#set-up-testing-in-ci\}

Es una buena idea probar tu aplicación de forma continua en un entorno controlado. No
importa qué método automatizado utilices, es fácil ejecutarlo con
[GitHub Actions](/testing/ci.mdx).
