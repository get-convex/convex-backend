Convex admite los siguientes tipos de valores:

<div className="convex-full-width">

| Tipo de Convex | Tipo TS/JS                                                                                                                | <div style={{width: "11em"}}>Ejemplo de uso</div> | Validador para [Validación de argumentos](/functions/validation) y [Esquemas](/database/schemas) | Formato `json` para [Exportación](/database/import-export) | Notas                                                                                                                                                                                                                                                                                                                                                   |
| -------------- | ------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Id             | [Id](/database/document-ids) ([string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)) | `doc._id`                                         | `v.id(tableName)`                                                                                                        | string                                                      | Consulta la sección [Document IDs](/database/document-ids).                                                                                                                                                                                                                                                                                              |
| Null           | [null](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#null_type)                                | `null`                                            | `v.null()`                                                                                                               | null                                                        | `undefined` de JavaScript no es un Valor de Convex válido. Las funciones que devuelven `undefined` o no devuelven nada devolverán `null` cuando se llamen desde un cliente. Usa `null` en su lugar.                                                                                                                                                      |
| Int64          | [bigint](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#bigint_type)                            | `3n`                                              | `v.int64()`                                                                                                              | string (base10)                                             | Int64 solo admite `bigint`s entre -2^63 y 2^63-1. Convex admite `bigint`s en [la mayoría de los navegadores modernos](https://caniuse.com/bigint).                                                                                                                                                                                                       |
| Float64        | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#number_type)                            | `3.1`                                             | `v.number()`                                                                                                             | number / string                                             | Convex admite todos los números de coma flotante de doble precisión IEEE-754 (como NaN). `Infinity` y `NaN` se serializan en JSON como cadenas.                                                                                                                                                                                                          |
| Boolean        | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#boolean_type)                          | `true`                                            | `v.boolean()`                                                                                                            | bool                                                        |
| String         | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)                            | `"abc"`                                           | `v.string()`                                                                                                             | string                                                      | Las cadenas se almacenan como UTF-8 y deben ser secuencias Unicode válidas. Las cadenas deben ser más pequeñas que el límite de tamaño total de 1 MB cuando se codifican como UTF-8.                                                                                                                                                                     |
| Bytes          | [ArrayBuffer](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer)              | `new ArrayBuffer(8)`                              | `v.bytes()`                                                                                                              | string (base64)                                             | Convex admite cadenas de bytes de primera clase, pasadas como `ArrayBuffer`s. Estas cadenas de bytes deben ser más pequeñas que el límite de tamaño total de 1 MB para los tipos de Convex.                                                                                                                                                              |
| Array          | [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)                          | `[1, 3.2, "abc"]`                                 | `v.array(values)`                                                                                                        | array                                                       | Los arrays pueden tener como máximo 8192 valores.                                                                                                                                                                                                                                                                                                        |
| Object         | [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#objects)                                | `{a: "abc"}`                                      | `v.object({property: value})`                                                                                            | object                                                      | Convex solo admite "plain old JavaScript objects" (objetos que no tienen un prototipo personalizado). Convex incluye todas las [propiedades enumerables](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Enumerability_and_ownership_of_properties). Los objetos pueden tener como máximo 1024 entradas. Los nombres de campo no pueden estar vacíos ni comenzar con "$" o "\_". |
| Record         | [Record](https://www.typescriptlang.org/docs/handbook/utility-types.html#recordkeys-type)                                | `{"a": "1", "b": "2"}`                            | `v.record(keys, values)`                                                                                                 | object                                                      | Los records son objetos en tiempo de ejecución, pero pueden tener claves dinámicas. Las claves deben ser solo caracteres ASCII, no estar vacías y no comenzar con "$" o "\_".                                                                                                                                                                             |

</div>
