Puedes exportar tus datos desde Convex
[realizando una copia de seguridad](/database/backup-restore) y descargándola como un archivo ZIP.

Alternativamente, puedes exportar los mismos datos con la
[línea de comandos](/cli.md#export-data-to-a-file):

```sh
npx convex export --path ~/Downloads
```
