Si te entusiasma el autoalojamiento, puedes ejecutar el backend de Convex en tus propios
servidores. La versión autoalojada de Convex ejecuta el
[backend de código abierto](https://github.com/get-convex/convex-backend) y
contiene el mismo código totalmente actualizado que utiliza el servicio en la nube.

Para comenzar con el autoalojamiento, sigue la guía de autoalojamiento:

  className="convex-hero-card"
  item={{

/>

Únete al canal `#self-hosted` en la
[comunidad de Discord](https://convex.dev/community) para obtener soporte sobre autoalojamiento.

El autoalojamiento no es para todo el mundo. Si buscas una solución más desatendida,
te recomendamos usar el
[producto alojado en Convex](https://convex.dev/pricing).

## Backend de Convex de código abierto \{#open-source-convex-backend\}

La mayor parte del backend está implementada en Rust, con una buena dosis de
TypeScript que da soporte al entorno de funciones del lado del servidor.

Puedes aprender más sobre cómo abrimos Convex como proyecto de código abierto en nuestro
[anuncio](https://news.convex.dev/convex-goes-open-source/) y en el
podcast [Software Engineering Daily](https://softwareengineeringdaily.com/2024/03/20/going-open-source-at-convex-with-james-cowling/).
Se distribuye bajo la [Licencia FSL Apache 2.0](https://fsl.software/).

## Otros proyectos de Convex de código abierto \{#other-convex-open-source-projects\}

El backend de Convex, las bibliotecas de cliente, el panel de control y la CLI son de código abierto.
Puedes explorar todo en la
[página de Convex en GitHub](https://github.com/get-convex).

### Clientes de Convex \{#convex-clients\}

Todos los clientes de Convex son de código abierto.

* [Cliente de Convex para JavaScript/TypeScript y la CLI](https://github.com/get-convex/convex-js)
* [Cliente de Convex para Python](https://github.com/get-convex/convex-py)
* [Cliente de Convex para Rust](https://github.com/get-convex/convex-rs)

### Mucho, mucho más \{#much-much-more\}

Convex también mantiene muchos otros proyectos de código abierto muy útiles, entre ellos
[helpers](https://github.com/get-convex/convex-helpers),
[templates](https://github.com/orgs/get-convex/repositories?type=all\&q=template),
[demos](https://github.com/get-convex/convex-demos), un
[entorno de pruebas](https://github.com/get-convex/convex-test) y mucho más. Consulta
la lista completa de todos nuestros repositorios públicos
[en GitHub](https://github.com/orgs/get-convex/repositories?type=all).
