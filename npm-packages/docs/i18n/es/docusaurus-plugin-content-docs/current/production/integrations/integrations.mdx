Convex se integra con una variedad de herramientas de terceros compatibles para
la transmisión de logs y el reporte de excepciones.

* [Log Streams](/production/integrations/log-streams) permite transmitir eventos
  de logs desde tu despliegue de Convex a destinos compatibles, como Axiom,
  Datadog o un webhook personalizado.
* [Exception Reporting](/production/integrations/exception-reporting) proporciona
  visibilidad de los errores en las ejecuciones de tus funciones de Convex.

## Configurar una integración \{#configuring-an-integration\}

Para configurar una integración, ve a
[Deployment Settings](https://dashboard.convex.dev/deployment/settings) en el
panel de control y a la pestaña &quot;Integrations&quot; en la barra lateral. Esta página proporciona una lista
de tus integraciones configuradas, su estado de funcionamiento actual y otras
integraciones disponibles para configurar. Para configurar una integración, haz clic en
la tarjeta y sigue las instrucciones de configuración.

![Integrations Page](/screenshots/integrations_page.png)

## Eliminar una integración \{#deleting-an-integration\}

Para eliminar una integración y dejar de enviar más eventos al
destino configurado, selecciona el icono de menú en la esquina superior derecha de un
panel configurado y elige &quot;Delete integration&quot;. Después de confirmar, la
integración dejará de ejecutarse en unos pocos segundos.

## Comentarios \{#feedback\}

facilitan el trabajo con servicios de terceros. Consulta la lista completa de
componentes en el [Directorio de Convex Components](https://convex.dev/components).
