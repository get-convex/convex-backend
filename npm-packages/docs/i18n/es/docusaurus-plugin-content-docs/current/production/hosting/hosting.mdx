La forma más sencilla de publicar tu aplicación web full‑stack es usar un proveedor
de alojamiento como [Vercel](https://vercel.com) o [Netlify](https://netlify.com).

Tanto Vercel como Netlify se integran con Git para desplegar código cada vez que se hace
push de una nueva revisión. Para alojar tu aplicación:

1. Haz commit de todos los archivos y haz push a tu proveedor de alojamiento Git favorito, como
   [GitHub](https://github.com/), [GitLab](https://gitlab.com/) o
   [Bitbucket](https://bitbucket.org/).

2. Sigue la guía correspondiente a continuación.

Si no usas Netlify o Vercel, puedes seguir la guía de alojamiento personalizado.

- [Vercel](/production/hosting/vercel.mdx)
- [Netlify](/production/hosting/netlify.mdx)
- [Alojamiento personalizado](/production/hosting/custom.mdx)
