  "Usa Convex directamente en HTML con etiquetas &lt;script&gt;, sin herramientas de compilación"
---

A veces solo quieres mostrar tus datos en una página web: sin instalar paquetes,
sin pasos de compilación y sin TypeScript. Suscribirte a consultas desplegadas en
un despliegue de Convex existente desde una etiqueta de script es sencillo.

import simpleHtml from "!!raw-loader!@site/../demos/html/simple.html";
import typedHtml from "!!raw-loader!@site/../demos/html/typed-example.html";
import script from "!!raw-loader!@site/../demos/html/script.js";

<Snippet source={simpleHtml} title="index.html" replacements={[[/https?:\/\/localhost:8000/g, "CONVEX_URL_GOES_HERE"]]} />

VS Code no admite la autocompletación de TypeScript en archivos HTML, así que, para disponer de tipos y un mejor autocompletado, puedes separar tu código en un archivo de script:

<Snippet source={typedHtml} title="index.html" replacements={[[/https?:\/\/localhost:8000/g, "CONVEX_URL_GOES_HERE"]]} />

<Snippet source={script} title="script.js" replacements={[[/https?:\/\/localhost:8000/g, "CONVEX_URL_GOES_HERE"]]} />

Consulta el [inicio rápido con etiqueta de script](/quickstart/script-tag.mdx) para obtener instrucciones sobre cómo configurar un nuevo proyecto de Convex.
