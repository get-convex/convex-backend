[Nuxt](https://nuxt.com/) es un potente framework web basado en Vue.

El paquete de npm
[`convex-nuxt`](https://www.npmjs.com/package/convex-nuxt), mantenido por la comunidad,
proporciona una integración estrecha de Convex con el ecosistema de Nuxt 3.

Utiliza [convex-vue](../vue.md) internamente.

Consulta el [inicio rápido de Nuxt](/quickstart/nuxt.mdx) para comenzar o la
[página de GitHub de convex-nuxt](https://github.com/chris-visser/convex-nuxt) para más
documentación.

[biblioteca `convex-nuxt`](https://github.com/chris-visser/convex-nuxt/tree/main)
