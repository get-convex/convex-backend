# Clientes JavaScript de Convex \{#convex-javascript-clients\}

Se puede acceder a las aplicaciones de Convex desde Node.js o cualquier entorno de ejecución de JavaScript que
implemente [`fetch`](https://developer.mozilla.org/en-US/docs/Web/API/fetch) o
[`WebSocket`](https://developer.mozilla.org/en-US/docs/Web/API/WebSocket). El
[Convex Client](/api/classes/browser.ConvexClient) reactivo permite que las aplicaciones web
y los servidores de Node.js de ejecución prolongada se suscriban a actualizaciones en las
consultas de Convex, mientras que el [cliente HTTP de Convex](/api/classes/browser.ConvexHttpClient)
se suele usar para renderizado del lado del servidor, migraciones, scripts administrativos
y funciones serverless para ejecutar consultas en un momento puntual.

Si usas React, consulta el
[`ConvexReactClient`](/api/classes/browser.ConvexClient) descrito en
[React](/client/react.mdx).
