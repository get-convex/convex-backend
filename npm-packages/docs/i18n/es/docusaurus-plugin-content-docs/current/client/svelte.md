Convex es una excelente opción para frameworks de UI reactivos como Svelte. El
[paquete convex-svelte de npm](https://www.npmjs.com/package/convex-svelte)
extiende [ConvexClient](/api/classes/browser.ConvexClient) con suscripciones
declarativas en Svelte 5.

Consulta la [guía de inicio rápido de Svelte](/quickstart/svelte.mdx) para comenzar. El
cliente de Svelte es de código abierto y está disponible en
[GitHub](https://github.com/get-convex/convex-svelte).
