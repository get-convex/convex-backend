[Vue](https://vuejs.org/) es un "framework accesible, con buen rendimiento y
versátil para crear interfaces de usuario web".

El paquete de npm
[`convex-vue`](https://www.npmjs.com/package/convex-vue) mantenido por la comunidad es una
biblioteca de integración de Vue.js para Convex. También se utiliza en
[convex-nuxt](./vue/nuxt.md).

Consulta la [guía de inicio rápido de Vue](/quickstart/vue.mdx) para comenzar o la
[página de convex-vue en GitHub](https://github.com/chris-visser/convex-vue)
para obtener más documentación.

<Admonition type="info">

La [biblioteca `convex-vue`](https://github.com/chris-visser/convex-vue) está
mantenida por la comunidad. Gracias al mantenedor
[Chris Visser](https://github.com/chris-visser) por su trabajo en este proyecto.

Puedes hacer preguntas sobre la biblioteca en el
[Discord de Convex](https://convex.dev/community), pero abrir un issue en
[convex-vue GitHub](https://github.com/chris-visser/convex-vue) es una mejor
manera de solicitar una nueva funcionalidad o informar de un error.

</Admonition>
