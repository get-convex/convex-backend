Ya sea que estés creando chatbots con RAG o una búsqueda rápida en tus aplicaciones,
Convex proporciona APIs fáciles de usar para crear productos potentes con IA y búsqueda.

[Vector Search](/search/vector-search.mdx) permite buscar documentos en función
de su significado semántico. Utiliza embeddings vectoriales para calcular la
similitud y recuperar documentos similares a una consulta dada. La búsqueda
vectorial es una parte clave de técnicas de IA comunes como RAG.

[Full Text Search](/search/text-search.mdx) permite búsquedas por palabras clave
y frases dentro de tus documentos. Admite coincidencias de prefijo para habilitar
la búsqueda mientras se escribe. La búsqueda de texto completo de Convex también
es reactiva y siempre está actualizada, como todas las consultas en Convex, lo que
facilita crear cuadros de búsqueda rápida confiables.

[Acciones de Convex](functions/actions) te permiten llamar fácilmente a APIs de IA,
guardar datos en tu base de datos y controlar tu interfaz de usuario. Consulta
ejemplos de cómo puedes usar esto para
[crear aplicaciones de IA sofisticadas](https://stack.convex.dev/tag/AI).
