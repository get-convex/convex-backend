Convex 让你能够轻松安排某个函数在未来一次性或周期性运行。这样你就可以构建持久的工作流，例如在用户加入一天后发送欢迎邮件，或定期与 Stripe 进行账目对账。Convex 提供了两种不同的调度功能：

* [Scheduled Functions](/scheduling/scheduled-functions.mdx) 可以由任何其他函数以持久方式安排在将来的某个时间点执行。你可以将函数安排在几分钟后、几天后，甚至几个月后运行。
* [Cron Jobs](/scheduling/cron-jobs.mdx) 会按固定周期调度函数运行，例如每天运行一次。

## 持久函数组件 \{#durable-function-components\}

内置的定时函数和 crons 非常适合较为简单的应用和工作流。
如果你在大规模场景下运行，或需要更严格的保障，请使用以下更高级别的
[组件](/components.mdx) 来实现持久函数。
