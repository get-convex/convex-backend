存储在 Convex 中的文件可以在
[变更函数](/functions/mutation-functions.mdx)、[操作函数](/functions/actions.mdx) 和
[HTTP 操作函数](/functions/http-actions.mdx) 中通过调用
[`storage.delete()`](/api/interfaces/server.StorageWriter#delete) 函数删除，
该函数接受一个存储 ID。

存储 ID 对应 `"_storage"` 系统表中的文档（参见
[元数据](/file-storage/file-metadata.mdx)），因此可以使用
`v.id("_storage")` 进行验证。
