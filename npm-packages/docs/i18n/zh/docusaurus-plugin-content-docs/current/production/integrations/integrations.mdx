Convex 可与多种受支持的第三方工具集成，用于日志流式传输和异常上报。

* [Log Streams](/production/integrations/log-streams) 支持将来自 Convex 部署的日志
  事件以流式方式发送到受支持的目标，例如 Axiom、Datadog 或自定义 webhook。
* [Exception Reporting](/production/integrations/exception-reporting) 让你能够清晰了解
  Convex 函数执行过程中的错误。

## 配置集成 \{#configuring-an-integration\}

要配置集成，请在仪表盘中进入
[Deployment Settings](https://dashboard.convex.dev/deployment/settings)，然后点击侧边栏中的 “Integrations” 选项卡。此页面会列出你已配置的集成、它们当前的运行状况，以及其他可供配置的集成。要配置某个集成，点击对应卡片并按照设置说明进行操作。

![Integrations Page](/screenshots/integrations_page.png)

## 删除集成 \{#deleting-an-integration\}

要移除某个集成并停止将后续事件发送到已配置的目标，点击已配置面板右上角的菜单图标，然后选择“Delete integration”。确认后，该集成会在几秒钟内停止运行。

## 集成组件 \{#integration-components\}

还能让你更轻松地对接第三方服务。你可以在
[Convex Components Directory](https://convex.dev/components) 中查看完整组件列表。
