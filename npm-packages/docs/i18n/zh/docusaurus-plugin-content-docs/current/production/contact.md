Convex 正在快速发展，我们始终期待你的反馈。

## 反馈与支持 \{#feedback-and-support\}

如果你有任何一般性问题、新功能需求或产品反馈，请在我们的
[Convex Discord Community](https://convex.dev/community) 中分享。我们尤其期待
看到你在 Convex 上构建的作品！

任何在我们的 Discord 频道中无法得到充分解答的具体支持问题，都可以发送邮件至
[support@convex.dev](mailto:support@convex.dev)。

## 关注 Convex \{#following-convex\}

版本更新说明会在 [Convex News](https://news.convex.dev/tag/releases)
和 [Convex Discord Community](https://convex.dev/community) 上发布。

产品公告、文章和演示会发布在
[Stack](https://stack.convex.dev/)、[News](https://news.convex.dev/)、[我们的 YouTube 频道](https://www.youtube.com/channel/UCoC_9mdiPwIu1sDxDtGQggQ)
以及 [X（原 Twitter）](https://x.com/convex) 上。

## 漏洞披露 \{#vulnerability-disclosure\}

如果你认为自己发现了 Convex 的安全漏洞，请通过
[security@convex.dev](mailto:security@convex.dev) 与我们联系，我们会在 24 小时内回复你。请在我们有机会解决该问题之前不要公开披露该问题。
