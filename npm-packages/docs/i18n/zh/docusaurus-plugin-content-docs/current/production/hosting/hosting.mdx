发布全栈 Web 应用最简单的方式是使用托管服务提供商，比如
[Vercel](https://vercel.com) 或 [Netlify](https://netlify.com)。

Vercel 和 Netlify 都可以与 Git 集成，在每次推送新版本时自动部署代码。要托管你的应用：

1. 提交所有文件，并推送到你常用的 Git 托管平台，例如
   [GitHub](https://github.com/)、[GitLab](https://gitlab.com/) 或
   [Bitbucket](https://bitbucket.org/)。

2. 按照下面对应的指南进行操作。

如果你不使用 Netlify 或 Vercel，可以参考自定义托管指南。

- [Vercel](/production/hosting/vercel.mdx)
- [Netlify](/production/hosting/netlify.mdx)
- [Custom Hosting](/production/hosting/custom.mdx)
