<span className="convex-hero">
  Convex Components 将代码和数据打包在一个沙箱中，让你可以自信且快速地为后端添加新功能。
</span>

Convex Components 类似于小型、完全自包含的 Convex 后端，安装它们始终是安全的。除非你显式地将应用的表或函数传递给它们，否则它们无法读取你的应用表或调用你的应用函数。

你可以在
[Convex: The Software-Defined Database](https://stack.convex.dev/the-software-defined-database#introducing-convex-components)
中了解整体愿景。

组件可以从 npm 或本地目录安装。安装后，它们拥有自己的数据库表和隔离的函数执行环境。你可以在
[Convex 网站](https://convex.dev/components)
上查看完整的组件目录。
