无论你是在构建支持 RAG（Retrieval-Augmented Generation）的聊天机器人，还是在应用中添加快速搜索功能，Convex 都提供了易用的 API，帮助你创建功能强大的 AI 与搜索能力。

[向量搜索（Vector Search）](/search/vector-search.mdx) 支持基于文档的语义含义进行搜索。它使用向量嵌入来计算相似度，并检索与给定查询相似的文档。向量搜索是 RAG 等常见 AI 技术的关键组成部分。

[全文搜索（Full Text Search）](/search/text-search.mdx) 支持在文档中进行关键字和短语搜索。它支持前缀匹配，从而实现输入联想（typeahead）搜索。与所有 Convex 查询一样，Convex 全文搜索具有响应式特性并始终保持最新状态，使你能够轻松构建可靠的快速搜索框。

[Convex Actions](functions/actions) 让你能够轻松调用 AI API、将数据保存到数据库，并驱动用户界面。查看如何使用它来[构建高级的 AI 应用](https://stack.convex.dev/tag/AI)。
