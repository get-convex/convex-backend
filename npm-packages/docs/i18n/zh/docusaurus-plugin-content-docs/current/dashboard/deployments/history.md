<Admonition type="info">

“历史”页面仅在 Convex Professional 中可用。

</Admonition>

此[“历史”页面](https://dashboard.convex.dev/deployment/history)是一个与配置相关事件的审计日志，用于记录在所选部署中发生的事件，例如函数部署、索引变更以及环境变量变更。

你还可以在[团队审计日志](/dashboard/teams.md#audit-log)中查看与团队相关事件的审计记录。
