[文件存储页面](https://dashboard.convex.dev/deployment/files) 会展示
[部署中存储的文件](/file-storage.mdx)。该页面还会显示文件的存储 ID、大小以及内容类型。你可以上传新文件，也可以下载或删除已有文件。

存储 ID 可能会被数据库中的文档引用。

<Admonition type="tip">

当新文件被上传后，UI 会显示最近上传的文件名。不过，这些名称不会被持久保存，当页面重新加载时将不再显示。

</Admonition>
