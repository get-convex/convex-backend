每个 Convex 项目都有一个主要的生产部署，你团队中的每位开发者还可以设置自己的个人开发部署。
另外，还有
[预览部署](/production/hosting/preview-deployments.mdx)，用于在将后端更改部署到生产环境之前进行测试。

在[部署页面](https://dashboard.convex.dev/deployment)上，你可以使用页面左上角的下拉菜单，在生产部署、你的开发部署以及任意预览部署之间切换。
