文件存储让你可以轻松在应用中实现文件上传，存储来自第三方 API 的文件并向其发送文件，以及向用户提供动态文件。支持所有类型的文件。

- [上传](/file-storage/upload-files.mdx) 文件，将它们存储在 Convex 中，并在数据库文档中引用
- [存储](/file-storage/store-files.mdx) 由第三方 API 生成或获取的文件
- 通过 URL [提供](/file-storage/serve-files.mdx) 文件
- [删除](/file-storage/delete-files.mdx) 存储在 Convex 中的文件
- 访问文件[元数据](/file-storage/file-metadata.mdx)

你可以在[仪表盘](/dashboard/deployments/files.md)中管理已存储的文件。

[使用 HTTP 操作的文件存储](https://github.com/get-convex/convex-demos/tree/main/file-storage-with-http),
[使用查询和变更的文件存储](https://github.com/get-convex/convex-demos/tree/main/file-storage)
