Convex 部署的公共接口由 `convex` 文件夹中各文件中定义的函数构成。

每个 Convex 部署的公共 HTTP 端点由通过 [HTTP Actions](/functions/http-actions.mdx) 定义的自定义 HTTP
端点以及一个静态的 [public HTTP API](/http-api/index.md) 共同构成。

部署还提供仅供该部署管理员使用的私有端点：

- [Streaming export API](/streaming-export-api.mdx)
- [Streaming import API](/streaming-import-api.mdx)
- [Platform APIs](/deployment-platform-api.mdx)
