要在 [React Native](https://reactnative.dev/) 中使用 Convex，请使用
[Convex React 客户端库](/client/react.mdx)。

请按照 [React Native 快速入门](/quickstart/react-native.mdx) 中的说明，完成 React Native 所需的特定配置。

你也可以克隆一个可运行的
[Convex React Native 示例项目](https://github.com/get-convex/convex-demos/tree/main/react-native)。
