Convex 非常适合与 Svelte 这样的响应式 UI 框架搭配使用。
[convex-svelte npm 包](https://www.npmjs.com/package/convex-svelte)
增强了 [ConvexClient](/api/classes/browser.ConvexClient)，使其在 Svelte 5 中支持声明式订阅。

请参阅 [Svelte 快速上手](/quickstart/svelte.mdx) 以开始使用。Svelte
客户端是开源的，可以在
[GitHub](https://github.com/get-convex/convex-svelte) 上获取。
