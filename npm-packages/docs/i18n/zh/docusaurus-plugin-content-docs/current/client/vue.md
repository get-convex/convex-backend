[Vue](https://vuejs.org/) 是一个“易上手、高性能且多功能的，用于构建 Web 用户界面的框架”。

由社区维护的
[`convex-vue`](https://www.npmjs.com/package/convex-vue) npm 包是一个为 Convex 提供的 Vue.js
集成库。它也为 [convex-nuxt](./vue/nuxt.md) 提供支持。

请参阅 [Vue 快速开始](/quickstart/vue.mdx) 以快速上手，或查看
[convex-vue GitHub 页面](https://github.com/chris-visser/convex-vue) 获取更多
文档。
