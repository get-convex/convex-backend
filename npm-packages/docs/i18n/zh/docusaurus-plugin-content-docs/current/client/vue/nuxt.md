[Nuxt](https://nuxt.com/) 是一个基于 Vue 的强大 Web 框架。

社区维护的
[`convex-nuxt` npm 包](https://www.npmjs.com/package/convex-nuxt) 为 Convex
与 Nuxt 3 生态系统提供了深度集成。

它在底层使用了 [convex-vue](../vue.md)。

请参阅 [Nuxt 快速开始](/quickstart/nuxt.mdx) 来上手，或查看
[convex-nuxt GitHub 页面](https://github.com/chris-visser/convex-nuxt) 了解更多文档。
