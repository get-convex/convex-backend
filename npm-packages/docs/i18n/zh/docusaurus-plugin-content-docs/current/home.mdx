Convex 是一个开源的响应式数据库，其中的查询就是直接在数据库中运行的 TypeScript 代码。就像 React 组件会对状态变化作出响应一样，Convex 查询会对数据库变化作出响应。

Convex 提供数据库、用于编写服务端函数的运行环境以及客户端库，帮助你轻松构建和扩展动态、实时更新的应用。
