Convex 让你可以轻松地通过在 JS 中运行的自动化测试或针对真实后端的自动化测试，以及在开发环境（dev）、预览环境和预发布环境中的手动测试来测试你的应用。

## 自动化测试 \{#automated-tests\}

### `convex-test` 库 \{#convex-test-library\}

[使用 `convex-test` 库](/testing/convex-test.mdx)，借助优秀的 Vitest 测试框架在 JS 中测试你的函数。

### 针对真实后端进行测试 \{#testing-against-a-real-backend\}

Convex 开源版本允许你在真实的[本地 Convex 后端](/testing/convex-backend.mdx)上运行并测试所有后端逻辑。

### 在 CI 中设置测试 \{#set-up-testing-in-ci\}

在受控环境中持续对你的应用进行测试是个好主意。无论你使用哪种自动化方法，都可以很方便地使用
[GitHub Actions](/testing/ci.mdx) 运行这些测试。
