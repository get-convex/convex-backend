Convex 支持以下类型的值：

<div className="convex-full-width">

| Convex 类型 | TS/JS 类型                                                                                                                   | <div style={{width: "11em"}}>示例用法</div> | 用于[参数验证](/functions/validation)和[模式](/database/schemas)的验证器 | `json` 中用于[导出](/database/import-export)的格式 | 备注                                                                                                                                                                                                                                                                                                                                                  |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------- | --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Id          | [Id](/database/document-ids) ([string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)) | `doc._id`                                        | `v.id(tableName)`                                                                           | string                                              | 参见 [Document IDs](/database/document-ids)。                                                                                                                                                                                                                                                                                                          |
| Null        | [null](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#null_type)                                    | `null`                                           | `v.null()`                                                                                  | null                                                | JavaScript 的 `undefined` 不是合法的 Convex 值。从客户端调用时，返回 `undefined` 或不返回任何值的函数会返回 `null`。请改用 `null`。                                                                                                                                                                             |
| Int64       | [bigint](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#bigint_type)                                | `3n`                                             | `v.int64()`                                                                                 | string (base10)                                     | Int64 值只支持介于 -2^63 和 2^63-1 之间的 BigInt。Convex 在[大多数现代浏览器](https://caniuse.com/bigint)中支持 `bigint`。                                                                                                                                                                                       |
| Float64     | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#number_type)                                | `3.1`                                            | `v.number()`                                                                                | number / string                                     | Convex 支持所有 IEEE-754 双精度浮点数（包括 NaN）。Inf 和 NaN 在 JSON 中会被序列化为字符串。                                                                                                                                                                                                                    |
| Boolean     | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#boolean_type)                              | `true`                                           | `v.boolean()`                                                                               | bool                                                |
| String      | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)                                | `"abc"`                                          | `v.string()`                                                                                | string                                              | 字符串以 UTF-8 格式存储，且必须是合法的 Unicode 序列。字符串在按 UTF-8 编码后，大小必须小于 1MB 的总大小限制。                                                                                                                                                                                                    |
| Bytes       | [ArrayBuffer](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer)                  | `new ArrayBuffer(8)`                             | `v.bytes()`                                                                                 | string (base64)                                     | Convex 原生支持字节串，作为 `ArrayBuffer` 传入。字节串在 Convex 类型中必须小于 1MB 的总大小限制。                                                                                                                                                                                                                |
| Array       | [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)                              | `[1, 3.2, "abc"]`                                | `v.array(values)`                                                                           | array                                               | 数组最多可以包含 8192 个值。                                                                                                                                                                                                                                                                                     |
| Object      | [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#objects)                                    | `{a: "abc"}`                                     | `v.object({property: value})`                                                               | object                                              | Convex 仅支持“普通 JavaScript 对象”（没有自定义原型的对象）。Convex 会包含所有[可枚举属性](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Enumerability_and_ownership_of_properties)。对象最多可以有 1024 个键值对。字段名必须非空，并且不能以 "$" 或 "\_" 开头。 |
| Record      | [Record](https://www.typescriptlang.org/docs/handbook/utility-types.html#recordkeys-type)                                    | `{"a": "1", "b": "2"}`                           | `v.record(keys, values)`                                                                    | object                                              | Record 在运行时是对象，但可以有动态键。键必须只包含 ASCII 字符、非空，并且不能以 "$" 或 "\_" 开头。                                                                                                                                                                                                               |

</div>
